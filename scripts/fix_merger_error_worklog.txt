Root Cause of TypeError:
The TypeError occurred in `scripts/ts_converters/lib/merger.ts` at line 45:
`const targetYMs = new Set(Array.from((targetTable as any).column('年月')).map(String));`

The error was: `TypeError: undefined is not iterable (cannot read property Symbol(Symbol.iterator))`

1. The target file didn't exist

3. The target table was empty (created with `aq.from([])`)

In these cases, the `column()` method returns `undefined` instead of an array, causing `Array.from()` to fail when trying to iterate over `undefined`.
